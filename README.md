# poc-dashboard

Dit project is een React + Vite demo die een financieel dashboard toont met grafieken en Tailwind CSS.

## Ontwikkeling

1. Installeer afhankelijkheden:
   ```bash
   npm install
   ```
2. Start de ontwikkelserver:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in de browser.

## Productiebouw

```
npm run build
```

Hierna kan je `npm run preview` gebruiken om de productiebuild lokaal te testen.
